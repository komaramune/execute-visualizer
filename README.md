# Execute Visualizer
https://komaramune.github.io/ExecuteVisualizer/

Minecraft Java Edition のexecuteコマンドの動作を3Dで可視化するWebアプリです。\
コマンド文脈の位置・向き・分岐を、サブコマンド分岐木と3Dビューで確認できます。

## 主な機能
- executeサブコマンドの構文解析と3D3Dビュアー上での可視化
	- 3Dビュアー上でのサブコマンドの位置・向きを表すロケーターの表示
	- セレクタ範囲の表示
- サブコマンド文脈木の表示
- エンティティごとの位置、向き、サイズ、視線高さの編集
- マクロ対応
- 現在の状態をJSONとしてクリップボードへセーブ、クリップボードからロード

## 補足情報
一部のサブコマンドや引数は未対応です。\
Minecraftのコマンドと動作が完全に一致するかは保証しません。\
このWebアプリはCodexを使ったバイブコーディングで開発されました。
